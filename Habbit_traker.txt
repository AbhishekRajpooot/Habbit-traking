Habbit trakker 

Problem Statement

Create a habbit traker app ,  where we can define habbits and track  them.  Create a fullstack app with Nodejs and Ejs

Features 
Add multiple habbits to track like reading book, going to the gym etc

Track each habit everyday. These are the 3 statuses of a habit:
Done - Mark the habit as done for a day
Not Done - Mark the habit as not done for a day
None - User did not take any action on a habit for a day 
A view to show all  current habits. Here give an add button where you can add a new habit to track
